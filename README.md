# Superstore Sales Dashboard - Power BI

This project analyzes Superstore sales data using Power BI.

## Dashboard Includes:
- Total Sales KPI
- Total Profit KPI
- Order Count KPI
- Sales Trend Over Time
- Sales by Category
- Profit by Category
- Sales by Region
- Interactive Slicers (Category & Region)

## Tools Used:
- Power BI
- Data Visualization
- DAX (Basic Aggregations)

This dashboard helps in understanding sales performance, profitability, and regional distribution.
